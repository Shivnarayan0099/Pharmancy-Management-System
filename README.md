# Pharmancy-Management-System


Menu-driven console application to manage medicine records digitally.
Uses Linked List for dynamic storage and Queue for purchase history.

## Features
- Add medicine
- Show medicines
- Search medicine
- Update medicine
- Purchase with billing
- Low stock alert
- Expiry check
- Dashboard

## Data Structures
- Linked List → store medicine records
- Queue → store purchase history (FIFO)

## How it works
User selects options from menu → program processes using linked list traversal → result displayed.

## Run Project

# Clone repo
git clone https://github.com/your-username/pharmacy-management-system.git

# Go to folder
cd pharmacy-management-system

# Compile
g++ main.cpp -o pharmacy

# Run
./pharmacy
